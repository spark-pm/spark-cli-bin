🚀 Get Started in 30 Seconds

Install SPARK CLI and start your first workspace:

1. Install

curl -fsSL https://cli.sparkpm.dev/install.sh | bash

2. Initialize Your Workspace

spark init

Supported platforms: Linux (x64), macOS (x64, arm64)
Windows users: Use WSL and follow Linux instructions

📋 Quick Start Guide

1. Initialize: spark init - Creates your workspace and guides you through setup
2. Create Task: spark task create "My first task" - Add work to your backlog
3. View Board: spark board - See your Kanban-style project board
4. Pull Work: spark task pull - Start working on a task
5. Get Help: spark --help - Discover all available commands
